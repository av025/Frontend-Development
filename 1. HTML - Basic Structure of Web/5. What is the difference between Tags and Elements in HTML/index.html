What was difference between html tags and html elements ?

HTML tags

To define our website content we use HTML tags

HTML tags are of two types

Opening tag and Closing tag
The html tags which has openinng tag and closing tag too.!!!!
For example h1 tag , p tag etc.
Self-closing tag or Empty tag
Tags which does not required closing tags for example hr tag, br tag , img tag , etc


HTML Elements

When we write content in our HTML tags mainly in Opening tag and Closing tag than entirely it was called as HTML Elements
