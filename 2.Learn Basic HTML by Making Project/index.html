Frontend Development

Frontend development is the development of the graphical user interface of the website, through the use of HTML, CSS, and Javascript, So that users can view and intereact with that website.

  • HTML : The Hypertext Markup Language or HTML is the markup language for documents designed to be displayed in a web browser.
  • CSS : Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML
