Introduction to Absolute Position

When we give any html element position Absolute than it was relative to nearest nont-static (Aka relative position or other position property mainly fixed) ancestor element.

When absolute position html element was not having non-static ancestor element than it was relative to root element of our html document.Absolute positioned elements are removed from the normal flow, and can overlap elements.
