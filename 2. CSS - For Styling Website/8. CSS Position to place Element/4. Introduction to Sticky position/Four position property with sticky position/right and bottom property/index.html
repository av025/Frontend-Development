How right and left property work with sticky position

When we give any html elemenet sticky position than it does not flow differnetly from flow of html document and sticky position was active on the scroll trigger.
So to use explcilty right and bottom property than we have to use container of relative position and than give child element sticky position.

So we assume that bottom and right property was default and our sticky position does not flow against the flow of html document like relative position and if scrolling activity done and it was mainly relative with viewport than we have to attached sticky element with parent element than we have to give overflow to that parent element Otherwise it was relative to viewport only.
