Introduction to Sticky Position

When we give any html element position sticky than it beomes fixed or Sticky type to that html element and it was the combination of relative and fixed position. Sticky position was also scrollable.

Combination of relative and fixed Position

When we give any html element sticky position so first it behaves like the relative position it does not flow against the normal flow of html documentation but when we start scrolling our webpage it was start scrolling with it like fixed position element but at certain level if sticky position element was relative to parent than it was scroll upto thier parent element or if the sticky element was relative to viewport than it was scroll at the last.
