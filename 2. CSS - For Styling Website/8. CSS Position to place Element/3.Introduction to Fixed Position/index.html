Introduction to Fixed Position

An element which has position fixed than it was relative to browser's viewport. Which means it remains at same place if the page was also scroll they was also scroll with page.

It behaves like absolute position but it was related to viewport and it does not flow in normal way in our html document and overlap on other html element too!!!

Let we see with example adding some dummy content in our html page

We give div position fixed than it become fixed to viewport and related to viewport and when we scroll our webpage this div element was also scroll.
