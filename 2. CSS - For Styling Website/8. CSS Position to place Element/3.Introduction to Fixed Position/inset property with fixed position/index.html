Inset property with fixed position

Inset was short-hand property to of four properties top, right, bottom and left. So we give all four position properties inside inset properties also together.

Example to show inset property

Giving position fixed to div element and also also give inset properties so here all the position we was giving calculated from viewport because whenever we give any html element fixed position than it becomes relative to viewport.

When we don't give width and height to any html element than after giving position if we apply two parallel position properties than left out space it was consider as width and height and we also control that with the help of giving max-width or max-height.
