Text Shadow Property in CSS

Similarly Box Shadow for giving shadow to box there was same working property was text-shadow which give shadow to our text. The syntax of text-shadow was first value for offset-x for x-axis and second value for offset-y for y-axis than third for blur radius to give blurieness for our text-shadow.
