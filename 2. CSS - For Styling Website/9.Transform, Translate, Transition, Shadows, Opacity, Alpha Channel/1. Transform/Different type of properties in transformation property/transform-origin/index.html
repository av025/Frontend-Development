Transfrom Origin Property

The transform-origin property allows you to change the position of transformed elements. The default origin of tranfrom property was from center and it was 50% from x-axis and 50% from y-axis. 2D transform elements we can change its origin from x-axis and y-axis. 3D transform elements can change it's origin from z-axis.

Example of giving transfrom-origin property to html elements

These div element we give transfrom property and default transform-origin property so it grow from center and it's cordinate from x-axis was 50% and from y-axis was 50%
Giving transfrom-origin property (0 0) it means x-axis was 0 and y-axis was 0.
