Translate transform property

Translate transform property helps to translate our html element in x-axis and y-axis or this property help to change the position of our html element in x-axis or y-axis. When we give translate property to any html element than it does not push other html element.

Giving percentage unit in translate property

When we give translate property in percentage unit that it takes position from equal to width and height. If we give translate property in x-axis than position was calculated from width and similarly if we give position in y-axis it calculated from height.
