Background-image

We use background-image property to give background-image for our html element.

With the help of background-image property we can add image in our html document. With the help of url( ' ') value we can add background-image.

Background-repeat property

With the help of background-repeat property we will control on our repeatation of background-image if we give value repeat-x than our background-image will repeat in x-axis and if we give repeat-y than our background-image will repeat in y-axis. or if we give no-repeat than our background-image will not repeat in any direction.

Background-size property

With the help of Background-size property we can give size of our background-image. the value's of background-size are .....

  • auto :- the default value of our background-image.
  • contain :- Contain background-size will fit our full image in our div element and it does not covered full width just fit full image so our background-image will not be covered full width anf height.
  • Cover :- Cover background-size will cover full background-image in our html element and it cut's our background-image edges. so when we give cover background-size than it fully background-image was not set if size was bigger.
  • Giving height and Width :- We can give width and height also but we prefer to give height and width in percentage unit and it spoilt our aspect ratio also.
