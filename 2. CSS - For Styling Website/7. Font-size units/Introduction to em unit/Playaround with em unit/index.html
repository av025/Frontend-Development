Play around with "em" unit

Set the font-size of body element was 30 px to learn and play around with em unit

Example one

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Saepe, ipsam. Sunt ex non blanditiis eos at enim, tenetur labore obcaecati.

We set body element font-size 30px and body element was also parent of all the html element we write inside body tag and we give example first font-size was 2em so 2 times of 30 was 60px so font-size of first example was 60px

Example Second

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Commodi, voluptate, perferendis numquam facere, quo vel qui veritatis dolorem excepturi consequatur quasi modi quae officiis?

We increase our font-size with the help of em unit similarly we can decrease the size of font-size too!!! When we give font-size in decimal value like 0.5em so we give second example font-size 0.5em so half of 30px was 15px that's how we decrease the font-size of second html element.
