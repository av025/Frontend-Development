Introduction to Font-size unit in css

We known that default font-size of every html element was 16px which given by user agent stylesheet (Aka default styling by browser ) except some html element has smaller font-size to 16px also for example button element

Basically Default styling was not 16px it was 1em

Two types of font-size units

  1. em unit
  2. rem unit

We understand these two font-size unit in depth.
