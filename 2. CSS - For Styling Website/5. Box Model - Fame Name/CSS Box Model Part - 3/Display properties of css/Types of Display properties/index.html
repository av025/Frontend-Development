Types of Display properties

There was mainly three types of display properties.

  • Block Display
  • Inline Display
  • Inline-block Display

With the help of Display property we can change display of any html element we want as per our need. Let we see example

Hello, Everyone Iam div element and also Block level element

When we give display inline than this div element was behave like inline element

Hi Everyone, Iam span element and also inline level element.

When we give display block to this span element than it behave's like block elemen

We can Manipulate Display of HTML Element with the help of Display property.
