Two categories of Inline element

There was two types of inline element

  • Non - replaced Inline element
  • Replaced Inline element

Non-replaced Inline element

The inline element in which we can write our content and than we can't give height and width , padding and border was also overlaping to another html element when we give in top & bottom direction and we can't give margin in top and bottom direction this is due to it's inline nature.

Example of Non-replaced Inline element

Span element was Non-replaced inline element anchor tag was also Non-replaced Inline element
em tag was also Semantic Non-replaced Inline element
