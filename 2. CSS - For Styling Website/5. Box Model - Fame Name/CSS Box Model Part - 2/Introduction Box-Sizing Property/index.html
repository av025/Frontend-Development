Here we give width = 500px and height = 200px to our html element but here was one problem it width and height was increases because of default nature of box-sizing. We give 10px border and 30px padding to it also so here width was claculated like that 10px (left-side border) + 30px (left-side padding) + 500 (width of content) + 30px (right-side ) + 10px (right-side border) = 580px
