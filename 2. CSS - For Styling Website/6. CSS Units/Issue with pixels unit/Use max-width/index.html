Using "max-width" to solve issue of Overflow

When we are using max-width than we are constrain or give limit that the html element will not grow more than some certain width. Here we always give aprroximately value.
max-width work if and only the width of html element was bigger than we are giving max-width value or if the width of html element was smaller of our max-width than it does-not work.
