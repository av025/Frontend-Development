Conclusion to Percentage unit

When we used percentage unit to any html element than every property was related to it's parent element property on what we are giving perecentage value.

  • When we give width & height in percentage unit to any html element than thier width and height was calculated to from there parents height and width.
  • When we give padding, margin in percentage unit than our padding and margin was also calculated from parent element width.
  • Percentage unit was not work with border property !!!!
  • On font-size this same concept apply but here our font-size value was calculated from parent's font-size.
