Viewport width

When we give width in viewport width {vw} unit of any html element than width relative to browser viewport width.

Example's of giving width in viewport-width unit (vw)

Giving width 10vw which means 10 percent of browser viewport size
Giving width 50vw which means 50 percent of browser viewport size
Giving width 80vw which means 80 percent of browser viewport size
Giving width 100vw means 100 percent of browser viewport size.

We can also called these unit viewport percentage width because here also we calculated width percentage wise of browser's viewport.
