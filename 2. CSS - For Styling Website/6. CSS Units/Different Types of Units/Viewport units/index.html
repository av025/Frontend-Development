Introduction to Viewport Units

This Viewport Units was related to browsers viewport width and height not like percentage unit relative to it's parent element. Viewport width helps to create responsive layouts for every device browsers screen.

Two types of viewport units

  • Viewport-width {vw} :- We use viewport width unit to give width to our html element.
  • Viewport-height {vh} :- We use viewport height unit to give height to our html element.
